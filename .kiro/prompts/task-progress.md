# Task Progress Analysis

Analyze the current task progress and provide recommendations.

## Instructions

1. Read `.kiro/specs/customer-management/tasks.md`
2. Identify completed tasks (marked with `[x]`)
3. Identify incomplete tasks (marked with `[ ]`)
4. Analyze dependencies between tasks
5. Identify parallelization opportunities

## Output Format

### Completed Tasks
- List all completed tasks with brief summary

### Next Recommended Task
- Which task should be done next and why

### Dependencies
- Which tasks depend on others
- Which tasks are blocked

### Parallel Opportunities
- Which tasks can be done simultaneously
- Suggested parallel groupings

### Estimated Effort
- Rough estimate for remaining work
